# Юридическая компания "Фемида" 

Веб-сайт юридической компании с полноценным личным кабинетом, каталогом услуг, системой заявок и административной панелью. Проект разработан с использованием классического стека PHP и MySQL.

## Основные возможности

### Публичная часть
- Информационные страницы: Главная, О нас, Области практики, Услуги, Юристы, Контакты.
- Интерактивный каталог юридических услуг с фильтрацией по категориям.
- Страницы профилей юристов с детальной информацией.

### Личный кабинет и авторизация
- Регистрация и вход пользователей.
- **Роль "Клиент" (user):** просмотр своих заявок, отслеживание статусов.
- **Роль "Специалист" (specialist):** редактирование своего профиля (включая загрузку фото), просмотр назначенных заявок, написание ответов клиентам и смена статусов.

### Административная панель
- Управление каталогом услуг (добавление, редактирование, удаление).
- Управление заявками: смена статусов и назначение ответственных специалистов.
- Статистика по сайту.

##  Технологический стек
- **Backend:** PHP 8+ (PDO, сессии)
- **Database:** MySQL 8.0
- **Frontend:** HTML5, CSS3 (кастомные стили, адаптивная вёрстка), JavaScript
- **Сервер:** Встроенный сервер PHP 

## 📦 Установка и запуск

### 1. Требования
- Установленный PHP (версии 8.0 или выше)
- Установленный MySQL Server
- Любая IDE (рекомендуется VS Code)

### 2. Настройка базы данных
1. Создайте пустую базу данных в MySQL:
   ```sql
   CREATE DATABASE themis_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
2. Импортируйте структуру таблиц из файла database/schema.sql.
3. Импортируйте тестовые данные из файла database/seed.sql.

### 3. Настройка подключения к БД
Откройте файл src/config/database.php и убедитесь, что там указаны правильные логин и пароль от вашего MySQL сервера:
$pdo = new PDO('mysql:host=127.0.0.1;dbname=themis_db;charset=utf8mb4', 'root', 'ваш_пароль', [
    PDO::ATTR_ERRMODE => PDO::ERRMODE_EXCEPTION
]);

### 4. Запуск проекта
## Способ А: Через встроенный сервер PHP (самый быстрый)
Откройте терминал в корне проекта и выполните команду:
php -S localhost:8000 -t public
Сайт будет доступен по адресу: http://localhost:8000

## Способ Б: Через OpenServer / XAMPP
Скопируйте папку public в директорию доменов вашего локального сервера.
Настройте виртуальный хост так, чтобы корневой папкой (Document Root) была именно папка public.
Откройте сайт в браузере.
